# 秩序尖塔 · ORDER TOWER — 设计文档

> 取材《Splatoon 3 · Side Order》的**类杀戮尖塔**肉鸽卡牌爬塔游戏。纯静态网页，零构建。

## 一、一句话

八号攀登一座把一切个性抹成纯白的「秩序之塔」，用**赤·蓝·黄·紫**四色卡牌构筑战斗方式，30 层、三名 Boss；登顶时的**调色盘配比**决定结局。

## 二、世界观 / 角色

- **八号**：主角，玩家扮演。
- **珍珠（无人机）**：随行吐槽与支援担当，多张黄/蓝卡牌与事件的来源。
- **玛丽娜**：塔外远程支援，全息终端事件、支援卡牌。
- **安可**：塔内隐居的音乐人，删卡/紫卡事件。
- **秩序之主**：塔顶元凶，真心认为「抹掉颜色是拯救」。

## 三、核心循环

```
序章 → 三幕（每幕 9 行分支地图 + Boss 层，共 30F）
  节点：战斗 / 精英(掉芯片) / 异变事件 / 商店 / 休息(回血或升级)
  战斗：回合制，墨水3 抽5，敌人意图预告
  胜利：彩币 + 三选一卡牌（芯片可+1），概率掉特饮
  幕间：回复 25% 生命 → 下一幕
登顶 → 按配色结算 6 结局之一；阵亡 → 「褪为纯白」结局
```

## 四、四色流派（= 结局评分轴）

| 色 | 流派 | 机制取向 | 单色结局 |
|---|---|---|---|
| **赤** | 火力 | 高伤、多段、大招爆发、残像复制 | 烈焰塔主 |
| **蓝** | 护体 | 墨甲、反墨、壁垒（甲不清零）、盾击 | 深海壁垒 |
| **黄** | 支援 | 抽牌、回费、回复、调色盘连携 | 辉光之翼 |
| **紫** | 侵蚀 | 毒、虚弱易伤、恶化翻倍、同化之刃 | 暗紫深渊 |

**同色共鸣**：四色各有「共鸣」卡，效果随本回合已打出同色牌数增强——单色爆发流 vs 均衡真结局（无单色 ≥45%）的取舍。

## 五、战斗系统

- 玩家 HP 72、墨水 3、手牌 5、手牌上限 10。
- 状态：墨力/灵巧/虚弱(-25%攻)/易伤(+50%受)/破损(-25%甲)/侵蚀(回合初掉血-1衰减)/反墨/净化层/仪式。
- 敌人意图预告实时折算墨力与虚弱/易伤修正。
- 能力牌：残像墨影（首攻复制）、墨之壁垒、秩序逆流、调色盘连携、墨孢之云、凝滞秩序等。
- 终 Boss：净化层 2 层开场，50% 血相位转换（清负面+墨力+4+25 甲）。

## 六、数值骨架

- 敌人：一幕小怪 8-24 HP / 精英 62-66 / Boss 95；二幕 26-42 / 84-88 / 150；三幕 36-64 / 115-120 / 215。
- 奖励：普通战 18-28 币、精英 42-60 币+芯片、Boss 95-110 币+三选一 Boss 芯片（能量+1 系）。
- 商店：卡 45-160 币、芯片 90-170、特饮 45-60、删卡 75 起每次+25。
- 前 4 行使用 easy 遭遇池，之后 hard；精英节点只出现在第 4 行以后。

## 七、平衡方法

用 Playwright 驱动的启发式机器人全流程自动打（威胁感知：甲不够先叠甲、否则集火最低血；随机走图含精英）：
- 调参目标：机器人胜率 ~10%（对应人类 30-60%），死亡分布覆盖三幕而非单点墙。
- 曾发现并修掉的墙：一幕 Boss 层数比 StS 提前一半 → HP 从 160 逐步压到 95；装甲蟹+歌者组合、双潜伏者等尖刺逐个磨平。

## 八、结局（6 种，localStorage 图鉴）

| 结局 | 触发 |
|---|---|
| 还世界以色彩（真结局） | 登顶 + 无单色 ≥45% |
| 烈焰塔主 / 深海壁垒 / 辉光之翼 / 暗紫深渊 | 登顶 + 对应色 ≥45% |
| 褪为纯白 | 阵亡 |

## 九、文件

`index.html` / `style.css` / `data.js`（全部数据与文案）/ `engine.js`（引擎）/ `ui.js`（渲染+WebAudio 音效）。改数值只需动 `data.js`。
