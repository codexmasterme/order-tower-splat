# 秩序尖塔 · ORDER TOWER

> 一款以《斯普拉遁 3》DLC「秩序篇（Side Order）」为世界观灵感的**类杀戮尖塔**肉鸽卡牌爬塔游戏。
> 纯静态网页，零依赖、零构建，同人致敬作品。

![类型](https://img.shields.io/badge/类型-Roguelike%20Deckbuilder-ff2d9c) ![平台](https://img.shields.io/badge/平台-浏览器-12e0d4) ![语言](https://img.shields.io/badge/语言-简体中文-f5c518)

## 🎮 怎么玩

**直接用浏览器打开 `index.html` 即可。** 无需安装任何东西。

```bash
# 或者起一个本地服务器（可选）
npx serve .
```

支持桌面与手机浏览器，进度自动保存（localStorage），关掉页面随时回来继续。

## 📖 故事

世界被塔顶的「秩序」漂成了纯白。八号带着调色盘醒来，珍珠无人机在耳边聒噪，玛丽娜在塔外远程支援——攀登 30 层秩序之塔，击败三名 Boss，把颜色还给世界。

**你的配色决定你的结局**：登顶时按牌组的赤/蓝/黄/紫配比结算，共 6 种结局可收集。

## 🗼 游戏内容

| 系统 | 规模 |
|---|---|
| 塔 | 3 幕 × 10 层 = 30F，每幕分支地图自由选路 |
| 卡牌 | 55+ 张：赤（火力）/ 蓝（护体）/ 黄（支援）/ 紫（侵蚀）四色流派 + 状态/诅咒 |
| 色彩芯片（遗物） | 21 枚，含 3 枚 Boss 专属芯片 |
| 特饮 | 8 种战斗道具 |
| 敌人 | 15 种小怪 + 6 名精英 + 3 名 Boss（终 Boss 双阶段） |
| 事件 | 10 个随机异变事件（含隐藏收益与风险抉择） |
| 节点 | 战斗 / 精英 / 异变 / 商店 / 休息（回血或升级卡牌） |
| 结局 | 6 种，图鉴收集制 |

## ⚔️ 核心机制

- **墨水**（能量）每回合 3 点，抽 5 张牌；**墨甲**（格挡）回合开始清零。
- 关键词：墨力 / 灵巧 / 虚弱 / 易伤 / 破损 / 侵蚀（毒）/ 反墨（荆棘）/ 净化层 / 消耗 / 虚无。
- **同色共鸣**：部分卡牌随本回合已打出的同色卡牌数增强，鼓励单色爆发流；与「均衡多彩」的真结局形成取舍。
- 敌人**意图预告**：头顶实时显示下回合行动与经修正的伤害数值。

## 🗂 文件结构

```
index.html   入口
style.css    样式（Side Order 苍白塔身 × 霓虹撞色美学）
data.js      全部数据：配置/卡牌/芯片/特饮/敌人/遭遇/事件/结局/文案
engine.js    引擎：地图生成/战斗系统/奖励/商店/事件/存档
ui.js        渲染：全部界面/覆盖层/音效（WebAudio 合成，无外部资源）
DESIGN.md    设计文档
```

## ⚖️ 平衡说明

数值经启发式机器人自动模拟校准（Playwright 全流程自动打牌，数十局采样）：贪心机器人胜率约 8-10%，对应人类玩家预期胜率 30-50%——有难度、但每种流派都能通关。所有数值集中在 `data.js`，欢迎自行魔改。
